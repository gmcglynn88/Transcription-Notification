How to Use – Queue-Based Risk Monitor

🔐 Authentication

OAuth implicit grant with Genesys Cloud (Ireland region). You'll be redirected to login.mypurecloud.ie.

📋 Queue Selection

Select one or more queues using the styled dropdown. The dropdown fetches real queues from the Genesys API using /api/v2/routing/queues.

⚠️ Risk Detection

The app polls the transcription notifications endpoint and scans for configured risk phrases. High severity: lawsuit, breach, fraud. Medium: complaint, cancel, supervisor. Low: other matches.
