How to use – Real‑time risk monitor

🔐 Authentication

OAuth implicit grant (same as blueprint). You’ll be redirected to login.mypurecloud.ie. After success, token is stored in session.

🎧 Start monitoring

Select work teams (optional), adjust risk keywords, and click "Start listening". The app opens a notification stream via polling the transcription notifications API (polling every 5s for demo). Each notification is scanned for risk phrases.

⚠️ Risk levels

High – words like "lawsuit, breach, fraud" Medium – "complaint, cancel, supervisor" Low – other matches. Hover shows phrase.

⚙️ Region & API

Fixed to Ireland (mypurecloud.ie). Uses the exact transcription notifications endpoint: https://api.mypurecloud.ie/api/v2/speechandtextanalytics/transcriptnotifications (simulated via demo data but fully structured).
