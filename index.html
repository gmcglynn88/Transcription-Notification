📘 Production Configuration Guide

Required Permissions

Ensure your OAuth client has these scopes:

  • speech-and-text-analytics - For transcription access
  • notifications - For creating channels and WebSocket connections
  • routing:queue:view - For fetching queue list

Transcription Requirements

  • Voice transcription must be enabled in your org
  • Enable per queue: /api/v2/routing/queues/{queueId}/transcription
  • Only Genesys Cloud Voice or BYOC Cloud provide real-time transcription
  • BYOC Premises completes transcription after call ends

API Usage Notes

  • Each notification counts toward your rate limits
  • Channels expire after 24 hours - use Refresh button to recreate
  • WebSocket will auto-reconnect with exponential backoff
